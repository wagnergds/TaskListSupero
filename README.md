# TaskListSupero #
Aplicativo Gerenciador de tarefas usando ASP.NET MVC

# # Visão geral

Este destreza Web foi criado como um exercício de codificação para mostrar habilidades usando MVC, C# e Entity Framework.

# # Motivação

A motivação para este projeto foi como um exercício de codificação para trabalhar para supero TI. Ele foi fornecido para aplicar nossas habilidades em C# e usando o padrão arquitetônico MVC.


# # Instalação

Para executar o projeto com êxito, você precisa baixar o Microsoft Visual Studio, que pode ser feito gratuitamente na Internet.

# # Características

Na pagina Inicial clique em Gerenciador de tarefas Supero

Como o usuário, você deve ser capaz de:

-Ver uma lista de tarefas, que são carregadas a partir de um banco de dados
-Clique no nome da tarefa para ver mais detalhes do mesmo
-Clique nas caixas de seleção para alterar o status
-Adicionar e criar novas tarefas
-Excluir qualquer tarefa
